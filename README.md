# TILT--lamp-that-obeys
A wearable / handheld tilt-controlled lighting system using an accelerometer and WS2812B LED strip.
There are lights that switch on.
There are lights that dim.

And then there is Tilt.

Tilt is not controlled.
It responds.

A small gesture, a slight movement of the hand,
and the light shifts — not abruptly, but like emotion.

Tilt was born from a simple question:
What if light could feel balance?

By embedding motion into illumination,
Tilt transforms orientation into expression.

A left lean might whisper red.
A right tilt might drift into blue.
Stillness settles into green — a moment of calm.

This is not just a lamp.
It is a conversation between gravity and glow.
